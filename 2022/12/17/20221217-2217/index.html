|阅读量:

安装

打开 https://store.rg-adguard.net/ ，复制https://www.microsoft.com/store/productid/9p3395vx91nr到搜索框，选择Slow搜索

复制地址到浏览器下载


双击安装比较小的安装包

把另一个比较大的安装包改名为wsa

右键点击「Windows 开始菜单图标」，点击「Windows 终端 (管理员)」，输入：

Add-AppxPackage + 你的wsa包路径

打开安卓子系统

下载 adb

配置adb环境变量



输入adb version查看环境是否配置成功

进入设置 → 应用 → 可选功能 → 更多 Windows 功能，找到并勾选开启Hyper-V和虚拟机平台两个选项，安装完成后会提示重启系统。


点击管理开发人员设置发现还是没有连接地址显示

找到一下解决办法，在bios界面开启虚拟化功能

参考：http://www.ujiaoshou.com/xtjc/170024817.html

重启之后已经正常显示

安装apk

adb install + 你的apk包路径
