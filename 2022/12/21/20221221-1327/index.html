使用兰空图床开源项目搭建自建图床

发表于2022-12-21|更新于2022-12-21|编程
|阅读量:

兰空图床版本：v2.1

前往 兰空图床 官网下载

前往 宝塔 官网输入服务器对应信息后，点击立即安装到服务器

版本选择：Nginx 1.20.2，MySQL 5.7.40，PHP-8.0.26，phpMyAdmin 4.4

登录宝塔面板，点击 网站 → 添加站点，填写域名信息

进入网站根目录，上传源码压缩包并解压，把压缩包下的所有文件剪切到根目录

进入宝塔软件商店 → PHP → 设置 → 安装扩展，安装fileinfo和imagemagick

进入宝塔网站 → 找到站点设置 → 网站目录，设置运行目录为/public，并取消勾选防跨域攻击

进入宝塔网站 → 找到站点设置 → 网站目录，设置伪静态如下：

进入宝塔网站 → 找到站点设置 → SSL → Let’s Encrypt，申请SSL证书，部署并关闭强制HTTPS


进入宝塔软件商店 → PHP → 设置 → 禁用函数，禁用readlink、symlink、putenv、getenv、exec、shell_exec、chmod、chown、fileperms，存在的话直接点击删除后，设置后记得重启PHP服务

在腾讯云设置域名解析

找不到index.php问题


进到网站的根目录，删除 composer.lock 文件，重新在根目录执行“composer install”，这样就能重新生成 composer.lock 文件了。
更换阿里云源

输入yes

输入yes

刷新页面可正常访问，填写好信息后点击立即安装出现以下错误


找到/www/wwwroot下的网站文件夹，将网站目录设置权限777即可正常访问

进入兰空图床首页，开启接口，关闭注册


创建游客组，删除超级管理员角色后重新添加超级管理员角色，在用户管理下面编辑超级管理员角色和容量

用Postman生成token实现Picgo第三方上传

token必须是：Bearer+空格接请求返回的token
